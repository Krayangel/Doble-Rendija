# Doble-Rendija

## Explicación del Experimento de la Doble Rendija:
# 1.
    - El experimento demuestra la onda de la materia
    - Cuando se envían partículas (fotones) una por una, crean un patrón de interferencia como si pasaran por ambas rendijas simultáneamente
        
# 2. Características
    - Si se observa por cuál rendija pasa cada partícula, el patrón de desaparece
    - Se puede desde el principio de que estas solo existen en superposición hasta ser observadas
        
# 3. Vision desde lo cuantico
    - Las partículas no siguen una trayectoria normal sino que parecen mas una onda
    - La función de onda representa las probabilidades de ubicación.
        
# Conclusión: 
    Este experimento nos muestra conflictos con la comprensión de la realidad física que estamos acostumbrados mostrando la naturaleza de la mecánica cuántica.
